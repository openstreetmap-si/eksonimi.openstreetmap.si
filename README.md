# Eksonimi v OpenStreetMap-u
Slovenski eksonimi v OpenStreetMap-u - https://eksonimi.openstreetmap.si

More about Slovenian exonyms in OpenStreetMap wiki: https://wiki.openstreetmap.org/wiki/Key:name:sl

Using:

1. [MapLibre GL JS](https://maplibre.org/projects/maplibre-gl-js/)
2. [OpenMapTiles vector tiles with `name:sl` tags](https://openmaptiles.org/languages/sl/)
3. [Klokantech language switching plugin for MapBox GL](https://github.com/klokantech/openmaptiles-language) adapted for MapLibre in [openmaptiles-language.js](docs/openmaptiles-language.js)
4. [OsmNames.org](https://osmnames.org) + [autocomplete widget](https://osmnames.org/api/)
5. ...more to come :)
